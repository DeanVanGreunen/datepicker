Tech Used:
    JS Libraries:
        jQuery | https://code.jquery.com/jquery-3.4.1.min.js

How To Use:
    There is an example.html file in each folder.

    Include:
        The js file and css file
            \minified version\*.[css|js]
        js including css
            \css included in js\*.js
        Dev build version.
            \dev build\*.[css|js]
    
    Create an input field with the datepicker attribute set to true, such as below

        <input type="text" datepicker="true" />
    

Notes:
    A unique datepicker is create for each datepicker input field.
